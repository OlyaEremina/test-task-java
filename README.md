## Утилита фильтрации содержимого файлов.
Консольная утилита для фильтрации текстовых файлов, содержащих вперемешку целые числа, 
строки и вещественные числа, и записи этих данных в разные файлы по типу.
Поддерживает обработку нескольких входящих файлов и набор опций командной строки.


## Требования
- Java: 21.0.10 (LTS);
- Система сборки: Apache Maven 3.9.12;
- Стронние библиотеки: не используются.


## Сборка проекта
В корневой директории проекта выполните: 

```bash
mvn clean package
```
После сборки JAR-файл будет создан в директории `target/` и скопирован в корневую директорию проекта.


## Запуск программы
Для запуска программы используйте команду:
```bash
java -jar util.jar [опции] <input1> <input2> ...
```

Пример запуска:
```bash
java -jar util.jar -f -p result_ -o some/mytest in1.txt in2.txt
```


## Описание опций
- `-p <prefix>` - задать префикс именам выходных файлов;
- `-o <path>` - указать директорию для сохранения результатов;
- `-a` - задать режим добавления в существующие файлы;
- `-s` - вывести в консоль краткую статистику: количество элементов, записанных в исходящие файлы;
- `-f` - вывести в консоль полную статистику по каждому типу отфильрованных данных.


## Особенности реализации
- Строки из файлов читаются по очереди в соответствии с их перечислением в командной строке, а в качестве 
  разделителя используется перевод строки;
- По умолчанию файлы результатов перезаписываются;
- Файлы с результатами создаются по мере необходимости, без заведомо пустых файлов;
- Если опции `-o` и/или `-p` указаны без значения, программа выводит предупреждение и продолжает работу,
  игнорируя эти опции;
- При указании опций `-s` и `-f` одновременно будет выведена только полная статистика;
- В случае ввода неизвестной опции программа выводит предупреждение и продолжает работу,
  игнорируя эту опцию;
- Если имена входные файлов не переданы, программа прекращает работу и выводит сообщение в консоль.
